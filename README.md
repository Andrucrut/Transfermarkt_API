## 📖 О проекте

Проект представляет собой **легковесный** и **удобный интерфейс** для сбора данных с сайта [Transfermarkt](https://www.transfermarkt.com/).  
Он использует технологию веб-скрапинга и предоставляет **RESTful API** на базе **FastAPI**.

Сервис позволяет разработчикам легко встраивать данные Transfermarkt в свои приложения, веб-сайты или аналитические пайплайны.


## ✨ Возможности

- 🔍 Сбор данных с Transfermarkt через RESTful API
- 🌍 **Получение списка национальных сборных**
- 👥 **Детальная информация об игроках в составах сборных**:
  - Позиция на поле
  - Возраст
  - Рыночная стоимость
  - Количество матчей и голов за сборную

> 💡 Это делает сервис полезным не только для анализа клубного футбола, но и для изучения международных турниров и составов национальных команд.



### Запуск через Docker
````bash

# Сборка Docker образа
$ docker build -t transfermarkt-api . 

# Запуск Docker контейнера
$ docker run -d -p 8000:8000 transfermarkt-api

# Открыть API в браузере
$ open http://localhost:8000/
````
